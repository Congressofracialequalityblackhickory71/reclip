## 🖥️ What reclip does

reclip is a lightweight media downloader with a clean web interface. It helps you save videos from many websites to your Windows computer. You open the app in your browser, paste a video page link, and start the download.

It is built for simple use on a home PC or a small local server. You do not need to learn command-line tools to get started.

## 🧰 Basic troubleshooting

### The app does not open

- Check that the download finished
- Make sure you extracted the ZIP file first
- Run the main app file again
- Restart your computer if the file still does not open

### The browser page does not load

- Close the app and open it again
- Wait a few seconds after starting it
- Check whether your browser blocked the local page
- Try a different browser

### The video will not download

- Confirm that the page link is correct
- Paste the full video page link again
- Try a different quality setting
- Make sure the website still allows downloads

### The file saves but will not play

- Try a different media player
- Check whether the file ended with the correct format
- Download the video again using a lower or higher quality
